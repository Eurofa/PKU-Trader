    上交所，市场代码 SHSE
    深交所，市场代码 SZSE
    中金所，市场代码 CFFEX
    上期所，市场代码 SHFE
    大商所，市场代码 DCE
    郑商所，市场代码 CZCE
    纽约商品交易所， 市场代码 CMX (GLN, SLN)
    伦敦国际石油交易所， 市场代码 IPE (OIL, GAL)
    纽约商业交易所， 市场代码 NYM (CON, HON)
    芝加哥商品期货交易所，市场代码 CBT (SOC, SBC, SMC, CRC)
    纽约期货交易所，市场代码 NYB (SGN)
